== Description ==

**Swift Coupons** is a powerful and flexible coupon plugin for WooCommerce that extends the default coupon system with advanced marketing tools. Create dynamic promotions using detailed qualifiers, flexible BOGO deals, shareable URLs, scheduling options, and auto apply â€” all designed to help you sell more.

**Key Features:**

- **Cart Qualifiers**: Create rule-based conditions using cart data, product data, user history, and more with AND/OR logic and condition grouping.
- **BOGO Promotions**: Offer Buy X Get X discounts with control over discount type (fixed, percentage, or override price).
- **Scheduling**: Schedule coupons by date, time, and day of the week.
- **URL Coupons**: Generate shareable links that apply coupons automatically.
- **Auto-Apply**: Automatically apply eligible coupons to the cart without user input.
- **Usage Restrictions**: Control coupon use based on user role, payment/shipping methods, and more.
- **Translation Ready**: Fully translatable and compatible with WPML and other localization tools.
- **Optimized for Performance**: Built to work with the latest WordPress and WooCommerce versions.

Whether you're running flash sales or personalized promotions, Swift Coupons gives you the control and flexibility to create high-converting campaigns.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/swift-coupons-for-woocommerce` directory, or install the plugin via the WordPress plugin directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Navigate to "Swift Coupons" in the WordPress admin menu to configure your coupon settings.

== Frequently Asked Questions ==

= Can I create Buy One Get One (BOGO) deals? =

Yes. You can create advanced "Buy X Get X" deals and customize how the discount applies (fixed amount, percentage, or override price). Options include matching quantity, applying the discount to the cheapest product, and allowing repeated use.

= What types of cart conditions can I configure? =

You can define conditions based on:
- Cart data like quantity, sutotal, etc
- Products and categories in the cart
- User data like order history, join time, etc
- Custom user/product meta
- And much more...

AND/OR logic and grouped conditions are also supported for complex rule sets.

= Are usage restrictions supported? =

Yes. You can restrict coupons by:
- Specific user roles
- Selected payment or shipping methods
- Logged-in status
- And other user or cart-based parameters

= Can I schedule coupons for specific time periods? =

Yes. You can:
- Set start and end dates and times
- Limit coupon use to specific days of the week
- Control hours of availability each day

= Can I share coupons via URL? =

Yes. Swift Coupons allows you to generate a unique URL that automatically applies a coupon when visited. You can customize the structure of the URL and the coupon code as needed.
